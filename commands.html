🛡️ Moderation
Command Description Example
+ban Permanently ban a member from the server +ban @user Spamming
+kick Kick a member from the server +kick @user Rule violation
+mute Timeout a member for a specified duration +mute @user 30m Spam
+unmute Remove a timeout from a member +unmute @user
+warn Issue a warning to a member +warn @user Bad behaviour
+warnings View all warnings for a member +warnings @user
+clearwarn Clear all warnings for a member +clearwarn @user
+purge Bulk delete messages from a channel +purge 50
+slowmode Set slowmode delay on a channel +slowmode 5s
+lock Lock a channel so only staff can send messages +lock #general
+unlock Unlock a previously locked channel +unlock #general
